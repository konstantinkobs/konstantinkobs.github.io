            💻 Projekte
            -----------
        
        Ich entwickle kleinere Software-Tools und Spiele und veröffentliche sie auf Github. Schon gemerkt? Der Header dieser Webseite generiert mit einem Hill-Climbing-Algorithmus gerade ein Bild in ASCII-Art. 😉

        - Für meine Oma habe ich bereits mehrere Spiele entwickelt, die sie auf ihrem iPad spielen kann, z.B.
            - SudoKOBS: Ein Sudoku mit täglich wechselnden Rätseln, basierend auf Github Workflows
            - Omagram: Ein Nonogram mit verschiedenen Schwierigkeitsstufen
            - BilderRESEL: Ein Bilder-Schiebe-Puzzle mit Familienbildern, die aber nur mit dem richtigen Passwort im Frontend entschlüsselt werden
        - kkcals: Ein Kalorientracker ohne Backend, basierend auf OpenAIs API
        - Rezepte-Lager: Eine digitale Rezepte-Sammlung für den schnellen Zugriff beim Kochen und Backen
        - Word2LinkedIn: Ein Konvertierungstool von Rich Text nach Unicode, um auch auf LinkedIn fette und kursive Texte zu schreiben
        - Glücksrad: Eine Drehscheibe, mit der Schülerinnen und Schüler die Stunde reflektieren können (entstanden aus dem Wunsch meiner Lehrerinnen-Ehefrau)

        Außerdem besitze ich einen 3D-Drucker und einen Laser-Gravierer, für die ich Designs entwickle und umsetze.
